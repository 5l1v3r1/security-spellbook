####Commands

```
    Core Commands
    =============

    Command       Description
    -------       -----------
    ?             Help menu
    help          Help menu
    clear         Clean the console
    exit          Exit the console
    quit          Exit the console
    search        Search for a module by name
    start         Start a module by name
 ```
